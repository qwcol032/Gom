<ul>

ul요소는 아이템의 순서가 중요하지 않은 목록을 나타냅니다.

<li>요소를 포함하고 있습니다.
대부분의 브라우저에서 목록앞에 블릿이 표현됩니다.

<ol>

ol요소는 순서가 있는 목록을 나타냅니다.

<li>요소를 포함하고 있습니다.
대부분의 브라우저에서 각 항목앞에 아라비아 숫자가 표현됩니다.

속성

  • reversed : 순서를 역순으로 만듭니다. 불리언 속성
  • start : 첫번째 목록의 순번을 지정해 줍니다.
  • type : 리스트 안에서 사용될 마커의 종류를 지정합니다.

<li>

부모 요소가 ol이라면 li요소는 순서있는 값을 가집니다.
부모 요소가 ul이라면 li요소는 순서가 없는 값을 가집니다.
