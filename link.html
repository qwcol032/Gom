<link>

link요소는 다른 리소스와의 연결을 위해 사용합니다.

주로 외부스타일시트나 대체문서 또는 작성자 지정하기위해 사용됩니다.
href, rel 속성은 반드시 존재해야 합니다.

속성

  • href : 링크의 목적지를 나타냅니다.
  • media : 어떤 매체에 적용되는지 나타냅니다.
  • rel : 링크된 대상의 관계를 나타냅니다.
  • type : MIME 타입을 나타냅니다.
