background

각 개별 배경 속성들을 한번에 지정하는 약식 속성입니다.

값

  • background-color
    • 색상값 : : #f00, #ff0000, rgb (250, 0, 0), red등으로 표현할 수 있습니다
    • transparent : 배경을 투명하도록 지정합니다.
  • background-image
    • uri : 배경 이미지로 지정할 이미지의 주소를 나타냅니다.
    • none : 배경 이미지를 지정하지 않습니다.
