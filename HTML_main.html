HTML이란 무엇인가?

html은 웹 문서를 만들기 위하여 사용하는 기본적인 프로그래밍 언어의 한 종류입니다.
특히 하이퍼텍스트를 작성하기 위해 개발되었으며, 인터넷에서 웹을 통해 접근되는 대부분의 웹 페이지들은 html로 작성됩니다.

html에서 사용하는 명령어는 태그(tag)라고 하는데 꺽은 괄호 <>를 이용하여 나타내며 일반적으로 태그의 시작은 <>을 사용하고 끝에 </>를 붙여 마칩니다.
하지만 <br>태그나 <img>태그와 같이 예외적으로 시작 태그만 사용하는 경우도 있습니다.

인터넷 발명이후 시간이 흐른뒤 html이 등장하였으며 그 이후에 html에 더 다양한 기능을 사용하고 쉽게 정리, 가독하기 위해 css, java script, php, jsp등이 등장하였습니다.
